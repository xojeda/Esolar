## Setup for EngelSolar VPN
### Initial Setup


    http://192.168.1.1
    user: admin
    password: admin01

   Log in and:
* Change password 
* Follow Wizzard, 
    change LAN IP to 10.10.x.1
    change Wifi Name and password
* Optional but recomended: Set all Devices with a fixed IP, i.e: 10.10.x.10, 10.10.x.11 and so.        
* Copy xxxxx.ovpn to router
`scp xxxxx.ovpn root@10.10.x.1:/etc/openvpn/xxxxx.ovpn`
* Copy root files to router
`cd ~/EngelSolar/modems/RU230`
`scp -r root/* root@10.10.x.1:/`
* ssh to router
`ssh root@10.10.x.1`

```
chmod 0755 /www/cgi-bin/status 
uci set openvpn.vpnname='openvpn'
uci set openvpn.vpnname.enabled='1'
uci set openvpn.vpnname.config='/etc/openvpn/xxxxx.ovpn'
uci set dhcp.lan.leasetime=1h
uci commit
/etc/init.d/openvpn restart
#Needed for the cgi interface
opkg update
opkg install openssh-sftp-server
```
